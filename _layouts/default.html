CafeHub

CafeHub is a platform that allows users to view cafes from around the country. Users can view all cafe locations, comment on a cafe, and even post their favorite cafes. This app was completed and deployed to Heroku on January 2017.

The biggest challenge for this project was working with MongoDB, setting up the config variables for production, and deploying to Heroku.

Tech stack: ExpressJS, MongoDB, Node.js, Google Maps Embed API, Bootstrap, Passport-local module

Discography

Discography is a web application that allows users to search for a collection of a particular artists discography and listen to a sample of a song. App completed December 2016 and deployed on github pages.

This was my first web application; the biggest challenge was using React Router to transfer data from the Spotify API while understanding React's modularized component structure.

Tech stack: React, React Router, Reflux, Spotify API, HTML/CSS, Javascript, jQuery

Movie Buff

MovieBuff is a web application that allows users to view a list of the top 100 movies. The app also permits users to post, edit and delete movies to the MovieBuff API. The application was completed and deployed April 2017.

The challenging aspect about this project was writing MySQL to create an ORM for the Movie Buff API and setting API headers to prevent CORS issues.

Tech stack: React, Redux, React Router, Express.js, MySQL, HTML/CSS
